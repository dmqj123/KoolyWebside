关于小工具

该文档匹配的Kooly版本：1.0.3

小工具其实就是通过Kooly快捷启动的程序，可以嵌入进Kooly使用。

你可以在“设置”中更改各个槽位的小工具。

你也可以通过点击槽位上小工具的图标快速启动小工具。

获取小工具

在小工具商城开放后，你可以在商城下载小工具。但目前，你只能通过其他途径获取小工具。当然，你也可以自己开发小工具

添加或删除小工具

在“设置”的小工具一栏，找到管理小工具，点击进入小工具管理界面，可以看到目前识别出来的小工具。

点击添加小工具，会自动打开小工具的文件夹。我们把要添加的小工具的文件夹拖进来，再进入小工具管理界面，点击刷新，看到新加的小工具即添加成功，此时可以将指定槽位的小工具替换为这个新的小工具。

删除也很简单，在小工具文件夹内，删除指定小工具的文件夹，即可删除小工具。

开发小工具

我们非常欢迎你自己开发想要的小工具，并在以后上架小工具商城。

其实我们只需要新建一个文件夹，里面放上你已经打包好的可执行文件和各种依赖文件，还要放上小工具的图标，再新建一个名为“exp.ini”的配置文件，编写配置文件就行了。

配置文件的编写：在第一行写上“program_path : ”加上可执行文件的名称，记得要加上英文的双引号；第二行写“icon_path : ”加上图标的文件名称，同样加双引号；第三行写“program_name : ”加上工具名称，同样加上双引号，并且要与文件夹名称相同。

示例文件：（exp.ini）(也可以参考内置小工具的)

                program_path : "tool.exe"
                icon_path : "icon.png"
                program_name : "Tool"

完成以上操作后，该文件夹就可以当成工具文件夹放进Kooly工具文件夹里，Kooly会自动扫描该工具。
